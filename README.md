## Running Locally
**Clone the repository the local machine.**
```
git clone https://github.com/unclebay143/cream-scanner
```

**Install the dependencies.**
```
pnpm install
```

**Run the application.**
Then, run the application in the command line and it will be available at http://localhost:3000.

```
pnpm run dev
```
